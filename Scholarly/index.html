Abstract

GaMa is a website that is desired for people all around the glob that wants to play different types of games, easy and fast without any sophisticated steps and last but not least, the registered clients will be able to register in tournaments for specific games, and see informations about their liked games from a library.

Introduction

As we all know, these days, the video games are gainning more and more popularity. Back in the day, people had to go in certain places to play games, but now the technology has advanced and we can do that from our computer and that's why the GaMa website is useful!

The purpose of this website is to:

  • Make it easy to play diffrent games with people all around the glob by joining tourneys.
  • All games are for different ranges of ages!
  • Easy to choose the game.
  • Easy to use
  • See all of your tournaments in one place.

Overview

The Technologies used

The entire Web Application is powered by PHP.

MySQL will be used for the data storage.

The Architecture

Procedural architecture we will see if we will change it

Below I present each page from the website and what it does

INDEX page is the page from where the user can log in or register. The user cannot see the games until it loggs in.

LOGIN page is the page from where the user can log in.

REGISTER page is the page from where the user can log in.

INDEX-AFTER-LOGIN page is the page from where the user can see the top 4 popular games for his age, he can get the RSS feed, and has a filter which is used to display games using ceratin criterias. He can see the Board/Digital games by genre, or if it has a tourney.

GAME-TOURNEY page is the page of a game. Here, the user can see details of the game, he can like the game or dislike it , he can also join a tourney if the game has it and if the tourney is not full. If the game has a tourney, the user will be able to see other people that are joined and their points. Each tourney has 8 slots. He can also export as CSV the details of the game. On this page, the admin has the rights to modify the number of players that are in the tourney and their points. He also has a button called "Battle" which makes the tourney to go at the next step. When wwe have a winner, it will apper a "RESET" button, which will reset the tourney and add a point in the database to the player that has won it.

MY-TOURNAMENTS is the page for the users where they can see the games that they like and the ones on which they are registered in the tourney.

ADMINISTRATION-PAGE this page can only be accesed by the admin. Here are 3 tables, the ones from the database. There are also 5 buttons: MOVE PLAYE it moves the player from a tourney to another. DELETE USER deletes the user from the database. ADD GAME adds a game to the website. DELETE GAME deletes a game from the website. MODIFY GAME modifies the details of the game(genre, age limit, type, tourney, description, dimensions).
